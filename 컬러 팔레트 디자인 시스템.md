## 🎨 컬러 사용 가이드

### 🟡 Primary Color (#F7D52A)
- **사용처**: 메인 배경, 히어로 섹션, 브랜드 로고 배경
- **효과**: 활기차고 즐거운 첫인상, 주목도 향상
- **주의사항**: 너무 넓은 영역에 사용하면 눈의 피로를 유발할 수 있음

### 🟤 Secondary Color (#8B4513)
- **사용처**: 헤딩 텍스트, 본문 텍스트, 네비게이션
- **효과**: 안정감과 신뢰성, 가독성 향상
- **조합**: 노란 배경 위에서 최적의 대비 효과

### 🟢 Accent Color (#4CAF50)
- **사용처**: CTA 버튼, 중요한 라벨, 하이라이트
- **효과**: 신선함과 건강함, 액션 유도
- **사용량**: 전체의 10% 이하로 포인트 사용

### 🤍 Background Colors
- **Primary (#F7D52A)**: 메인 섹션, 히어로 영역
- **Secondary (#FFF8DC)**: 카드, 모달, 서브 섹션
- **White (#FFFFFF)**: 텍스트 배경, 클린한 영역

### 📝 Text Hierarchy
```css
/* 메인 헤딩 */
h1 { color: var(--color-text-primary); }

/* 서브 헤딩 */
h2, h3 { color: var(--color-text-primary); }

/* 본문 텍스트 */
p { color: var(--color-text-primary); }

/* 강조 텍스트 */
.highlight { color: var(--color-accent); }

/* 보조 텍스트 */
.muted { color: var(--color-text-muted); }
```

## 📱 반응형 고려사항

### 다크모드 지원
```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #1A1A1A;
    --color-bg-secondary: #2D2D2D;
    --color-text-primary: #F7D52A;
    --color-text-secondary: #FFFFFF;
    --color-primary: #FFF176;
  }
}
```

### 모바일 최적화
```css
@media (max-width: 768px) {
  :root {
    /* 모바일에서는 더 부드러운 톤 사용 */

}
```
